# recipe-note

Hong Company의 ver 1.0 버전 recipe를 기록할 수 있는 노트앱 입니다.

## Installation

`Clone project` :

    git clone https://github.com/BrendenHJH/recipe-note.git

`현재 위치로 이동` :

    cd recipe-note
    

## 프로젝트 제약사항

* Back-end 요구사항
    * 필수사항
        * java 8
        * springboot 2.0
        * spring data JPA
        * H2
        * gradle, lombok
    * 선택사항
        * multi module project
        * QueryDSL
        * Spring REST Docs
        * Spring Security
* Front-end 요구사항

* 배포 요구사항

## 개발 요구사항

* github를 통한 이슈 관리
